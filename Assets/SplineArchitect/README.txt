README.txt
================================
Spline Architect is a powerful editor tool for the Unity game engine.
It consists of a base version and several optional addons that can be purchased separately.

Quick Start:
1. Open 'Installation.pdf' for setup.
2. See 'User manual.pdf' for usage instructions.
3. Explore the 'Examples' folder to see ready-made scenes.

Support:
